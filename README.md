# 💖 موقع SARA — Love Website v2.3

موقع رومانسي متكامل **محمي بباسوورد** + صورة بروفايل سحرية + موسيقى تلقائية ✨

---

## 🆕 الإضافات الجديدة:

### 📸 صورة البروفايل السحرية
بعد ما تدخلي الباسوورد، صورتك بتظهر بشكل سحري في شاشة الترحيب:
- إطار ذهبي/وردي **بيدور**
- 3 حلقات متوهجة بتلف حواليها
- نجوم بتدور في مدارات
- قلوب صغيرة بتطفو حواليها

### 🎵 الموسيقى تشتغل تلقائياً
بمجرد ما تدخلي الباسوورد الصح، الموسيقى تبدأ تشتغل تلقائياً.

---

## 🎯 الإضافات السريعة (3 خطوات):

### 📸 1. إضافة صورة البروفايل (السحرية)

ضعي صورتك في:
```
sara-love-website/public/images/
```

**سميها بالاسم ده بالظبط:**
- `profile.jpg`

💡 **نصائح:**
- استخدمي صورة بورتريه شكل مربع
- الأبعاد المثالية: 500x500 بكسل
- حجم أقل من 1 ميجا

✨ هتظهر تلقائياً في شاشة الترحيب!

---

### 📸 2. إضافة صور المعرض

في نفس المجلد `public/images/`، ضيفي:
- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`
- `photo4.jpg`
- `photo5.jpg`
- `photo6.jpg`

✨ هتظهر تلقائياً في قسم المعرض!

---

### 🎵 3. إضافة الموسيقى

ضعي الأغنية في:
```
sara-love-website/public/music/
```

**سميها بالاسم ده:**
- `our-song.mp3`

✨ هتشتغل تلقائياً بعد ما تدخلي الباسوورد!

## 🚀 طريقة التشغيل:

### الخطوة 1: ثبتي Node.js
من https://nodejs.org (LTS)

### الخطوة 2: افتحي المشروع في VS Code

### الخطوة 3: ثبتي المكتبات
```bash
npm install
```

### الخطوة 4: شغّلي الموقع
```bash
npm run dev
```

هيفتح على: **http://localhost:3000**

## ⚠️ ملاحظات مهمة عن التشغيل التلقائي للموسيقى:

المتصفحات الحديثة (Chrome, Safari, Firefox) أحياناً بتمنع تشغيل الموسيقى التلقائي عشان حماية المستخدم.

✅ **الحل تلقائي:** بما إن الباسوورد بيتدخل من المستخدم، ده بيعتبر "تفاعل" والمتصفح بيسمح بالتشغيل بعد كده.

⚠️ **لو المتصفح منع التشغيل** (نادر): هيظهر تلميح صغير في الأسفل: "دوسي على زر التشغيل عشان تسمعي أغنيتنا".

## ❓ حل المشاكل:

### الصورة مش بتظهر في شاشة الترحيب؟
- تأكدي من الاسم: `profile.jpg` (مش `Profile.jpg` أو `profile.JPG`)
- تأكدي من المسار: `public/images/profile.jpg`
- جربي Refresh الصفحة

### الأغنية مش بتشتغل تلقائياً؟
- ده طبيعي في بعض المتصفحات
- هتظهر رسالة بسيطة تطلب من المستخدم يضغط زر التشغيل

### `ENOSPC: no space left on device`
فضّي مساحة على القرص.

### `ERESOLVE`
```bash
rm -rf node_modules package-lock.json
npm cache clean --force
npm install
```
